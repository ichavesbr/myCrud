# CRUD App de Usuários


## Descrição
O projeto desenvolvido com foco em aprendizado de backend e integração com frontend moderno.

---

## Funcionalidades
- Autenticação básica (login)
- CRUD de usuário (nome, idade)
- Conexão com banco MySQL
- Estrutura organizada de backend e frontend

---

## Tecnologias
- **Frontend:** React, JavaScript, Axios (para chamadas HTTP)
- **Backend:** Node.js, Express, bcrypt, JWT
- **Banco de dados:** MySQL
